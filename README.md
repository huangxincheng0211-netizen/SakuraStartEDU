# 樱启教育官网简易版

这是一个最简易的一页式静态网站，适合用于课程宣传、咨询转化和手机端浏览。

## 文件说明

- index.html：网页主体内容
- style.css：页面样式
- script.js：复制微信号功能

## 使用方法

直接双击打开 `index.html` 即可预览。

## 后续可替换内容

1. 将二维码区域替换成你的真实微信二维码图片。
2. 可以继续加入机构 logo、课程封面、学员案例、价格表。
3. 可以部署到 GitHub Pages、Vercel、Netlify 或服务器。
